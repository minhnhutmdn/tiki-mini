# TIKI MINI API

###Getting Started###

Then start process with the following:


INSTALL
```
    > cd tiki_api
    > npm install
```

MONGODB IMPORT DB
```
	> mongorestore -d tiki-product-api FULL-PATH-DIRECTORY/tiki_api/tiki-product-api
	> mongod

```

START SERVER
```
	> npm run dev
```


# TIKI MINI


```
	> npm install
	> npm start
```
