# TIKI MINI API

###Getting Started###

Please start process with the following:

INSTALL
```
	> npm install
```

MONGODB IMPORT DB
```
	> mongorestore -d tiki-product-api ~/tiki_api/tiki-product-api
	> mongod

```

START SERVER
```
	> npm run dev
```

API DESKTOP TOOLS
```
Postman, Robomongo

```


###Have Fun!!!###
